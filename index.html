Projects

COGNITEE Website

This project was to create a brand new website both design and development for the company COGNITEE based in Tokyo, Japan.

The website was designed in Adobe XD for wireframing and mockups of the website. After the design was approved, I was tasked with developing the website in VueJS.

COVID PREVENTION Game

This project was to design and develop a game that would help inform players about COVID-19, with safe practices and such.

The game was designed and developed as a group for a course module. Designed through Microsoft Teams group meetings and developed in Unreal Engine 4.

AKIRA FUKU Website

This is a future project created by myself as a passion project to open up a streetwear brand based on japanese streetware, and bring it into UK.

The website will be designed in Adobe XD for the wireframing and mockups of the website. Once designs are finalised the website will be developed in React.

Inventory Management System (IMS)

This project was to create an Inventory Management System in Java with the use of SQL for the database off a template.

The project was coded in Java and MySQL was used to create the database portion. The user could add, update, and delete multiple entities on the system such users, orders, and items.

Hobby Project

This project was to create a hobby website that uses Java and HTML / CSS. The theme behind the website was streetware.

The website was designed and developed with Java and Spring for the back-end, while the front-end was developed in HTML and CSS. The user would be able to add and remove clothing and brands on the website.

QA Cinema Website

This project was a team project created to design and develop a cinema website under a specification provided by the company.

The website was created with the MERN stack; MongoDB, Express, ReactJS and NodeJS. My role for the project was to code the front-end of the website to make it look visually appealing.
