Projects

COGNITEE Website

This project was to create a brand new website both design and development for the company COGNITEE based in Tokyo, Japan.

The website was designed in Adobe XD for wireframing and mockups of the website. After the design was approved, I was tasked with developing the website in VueJS.

COVID PREVENTION Game

This project was to design and develop a game that would help inform players about COVID-19, with safe practices and such.

The game was designed and developed as a group for a course module. Designed through Microsoft Teams group meetings and developed in Unreal Engine 4.

AKIRA FUKU Website

This is a future project created by myself as a passion project to open up a streetwear brand based on japanese streetware, and bring it into UK.

The website will be designed in Adobe XD for the wireframing and mockups of the website. Once designs are finalised the website will be developed in VueJS.
